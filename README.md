# Afrotorget

An Afro-Norwegian community platform combining a news portal, business directory, and marketplace — built for the African diaspora in Norway.

## What it does
- **News/Blog** — articles and stories from the Afro-Norwegian community  
- **Business Directory** — searchable listings of African-owned and African-focused businesses in Norway
- **Marketplace** — product listings with categories and adverts
- **Ratings & Bookmarks** — users can rate businesses and save favourites

## Tech Stack
- Ruby 2.7 / Rails 6.1
- PostgreSQL
- Devise (authentication)
- Bootstrap + Font Awesome
- CircleCI (CI/CD)
- Rubocop (code quality)

## Status
Early-stage MVP. The core domain model (10 models covering users, companies, products, blogs, bookmarks, ratings) is complete. Frontend styling is in progress.
